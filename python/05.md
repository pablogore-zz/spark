# While 
```python
    i=2
    while i>12
        print(i)
        i+=3
```

# Python condicionales
```python
    
    for i in range(1,10):

```

```python
    
    for i in range(0,5):
        if i % 3 == 0 :
            print(i)
        elif i % 3 == 1 :
            print(i + 10)
        else:
            print(i - 10)
```
